## Priority tiers (from baseline worksheet)

**A — Rescue (comfort 1–2):** Darkness, Pretender, Lola Montez, Valerie, Voodoo Child  

**B — Form (comfort ~3):** Thunderstruck, Don't Stop Believin', Hunger Strike, Teenage Dirtbag, Fat Bottomed Girls, Mama, Ain't Goin' Down  

**C — Fills (use cheat Fills / More fills):** Sweet Home, Purple Rain, Dream On, Mary Jane, Banditos, Zombie, Ain't Goin' Down  

**D — Maintenance (≥4):** cool-down / set glue only

## Phase defaults for Priority + Slice

### Phase 1 (through Jul 20) — Rescue

- **Priority (45):** Darkness + Pretender every day; rotate Lola / Valerie / Voodoo into the remaining time.
- **Slice (50):** rotate kill zones — `1–6` · `17–22` · `23–27` · `23–31` · diagnostic A-only.

### Phase 2 (Jul 21–26) — Form + fills

- **Priority (45):** Mon/Wed/Fri form (B); Tue/Thu/Sat fills (C). Keep 10–15 min Darkness+Pretender if still &lt;3.
- **Slice (50):** `7–16` · `1–10` · `11–20` · `17–26` · `21–31` · `1–16`.

### Phase 3 (Jul 27–31) — Harden

- **Priority (45):** weakest from notes + changeover pairs.
- **Slice (50):** alternate `1–16` and `17–31`; Jul 30 = abbreviated full set (V+C on ≥4, full on ≤3, real Drop D retunes).

### Show day (Aug 1)

- ~25 min: songs 1–4, Pretender entrance, Middle + Fat Bottomed tuning check. No learning.

### Micro-blocks (pick one per gap)

1. **Isolate (30–45s):** B→Bm→F#→F#m, F→Bb, Am/Em barre @2 and @4. Name silently.
2. **Neighbor changes (45–60s):** E→B→E, A→Bm→A, E→F#m→E, Am→Dm→Am, F→Bb→F.
3. **Power root walk (30s):** E-string F5→A5; A-string Bb5→D5. Optional: Thunderstruck E5–B5 / F#5 moves (Pretender bridge **A5** only — not the old A5–F5–C5–G5 wall).
4. **Blues 7ths (30s):** E7 A7 B7 D7 C7.
